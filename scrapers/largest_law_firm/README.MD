## Getting Started

###### Prerequisites

`python3` installed in your system

###### Installation

1. Clone the repo (if you don't have yet in your computer)
   ```
   git clone https://github.com/deepjudge-ai/scrapers.git
   ```
2. Install the dependencies
   ```
   pip install -r requirements.txt
   ```
3. Run .py file
